# Practical Machine Learning: Course Project

This repository contains the R Markdown and HTML files for the Coursera Practical Machine Learning course project (Johns Hopkins University).

### Project Overview
The goal of this project is to predict the manner in which six participants performed barbell lifts. This is the `classe` variable in the training set. Data was collected from accelerometers on the belt, forearm, arm, and dumbbell.

A Random Forest model was trained using 3-fold cross-validation to predict whether the exercise was performed correctly (Class A) or with one of four common mistakes (Classes B-E).
